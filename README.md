# gapjunction

Code for simulation of Rz cell gapjunction of leech.

### Parameters from real recordings:
Given Retzius cells A and B connected by a gapjunction:

R<sub>A</sub> = 23 MΩ,
C<sub>A</sub> = 1000 pF,
τ<sub>A</sub> = 24 ms

R<sub>B</sub> = 34 MΩ,
C<sub>B</sub> = 500 pF,
τ<sub>B</sub> = 19 ms
